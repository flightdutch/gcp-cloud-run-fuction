Створіть Секрет у GitHub
    В Google Cloud Console створіть Service Account з роллю Editor (або Cloud Run Admin + Storage Admin).
    Створіть для нього JSON ключ і завантажте його.
    У GitHub репозиторії перейдіть у Settings -> Secrets -> Actions і додайте новий секрет:
        Назва: GCP_SA_KEY
        Значення: скопіюйте туди весь вміст вашого JSON файлу.

Переконайтеся, що ваш Service Account має ролі:
    Cloud Run Admin / Cloud Run Developer — для керування сервісами Cloud Run.
    Service Account User — щоб GitHub міг запускати процеси від імені сервісного акаунта.
    Storage Admin
    Artifact Registry Administrator — якщо ви збираєте власний Docker-образ.
    Cloud Build Editor / Cloud Functions Developer — якщо ви використовуєте саме Cloud Run Functions.

Увімкніть API:
    gcloud services enable run.googleapis.com cloudbuild.googleapis.com
    gcloud services enable run.googleapis.com artifactregistry.googleapis.com
